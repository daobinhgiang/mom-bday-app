# UI Kit — Ngôi Nhà Của Mẹ (App)

Interactive recreation of the app: door → hub → rooms → book/song/postcard interactions → farewell.

## Files
- `index.html` — full prototype, single-file. Open this to see the app.
- `README.md` — this file.

## What's recreated
- **Door (login)** — knock-to-enter, opens with hinge physics
- **Hub** — painted house cross-section with hot zones over each room
- **Library** — bookshelf, click a book to open spread
- **Music room** — turntable + dedicated songs, lift/swing/drop arm
- **Map room** — pins (lived / visited / dream)
- **Family room** — wood-frame photo wall, slight tilt
- **Bedroom** — Mom's writing desk
- **Garden** — grandkids' contributions
- **Toolbar** — mute, time-of-day cycle, farewell
- **Farewell** — dimming amber→indigo wash, single lantern, "Hẹn gặp Mẹ ngày mai."

## What's stubbed
- Sound is a UI affordance only — actual audio files not bundled (see brief: real recordings, not synthesized).
- Painted illustrations are SVG scaffolds. Production must replace with commissioned watercolor.
- Weather + holiday overlays not implemented in this kit (Tết, Vu Lan, rain) — documented in README.

## Source of truth
The original repo (`daobinhgiang/mom-bday-app`) is a placeholder dark-blue/gold layout that doesn't match the design brief. This kit was built from the **brief**, not the placeholder.
